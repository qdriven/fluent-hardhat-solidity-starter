# hardhat intro

- Install
```shell
npm install -g hardhat
```
- Create Project: WSL2
```shell
npx hardhat 
```

- hardhat tasks list

```sh
npx hardhat
```

- compile

```sh
npx hardhat compile
```

- test 

```sh
npx hardhat test
```

- test with coverage

```sh
npx hardhat coverage
```
- test with  gas report

```sh
REPORT_GAS=true npx hardhat test
```
